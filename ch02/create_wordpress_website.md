# 架設 WordPress 網站

## 需求

在 Microsoft Azure 上架設 WordPress 網站。

## 事前準備

  * 初步認識 Microsoft Azure 應用程式服務。
  * 瞭解如何[架設靜態網站](create_a_static_website.md)。

## 解決方法

### 建立 WordPress 服務

進入 Microsoft Azure （預覽版）[管理後台](https://portal.azure.com/)，在左側的面板中找到 **新增** 的按鈕，然後選擇 _Marketplace_。
